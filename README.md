# Welcome to Hansa Tharuka Live!

This is the starter package for firebase database with node js Express server.
simply create a Firebase database project and add 

apiKey:  "",
authDomain:  "",
databaseURL:  "",
projectId:  "",
storageBucket:  "",
messagingSenderId:  "",
appId:  ""

above fields.

run **npm install** 
run **node server.js**
